# AI Video Generator
AI-powered YouTube video generator with voice, images, and auto-script

## Created by Fiaz Shah

## Features
- AI Script Generation (Groq)
- Male Voice (Christopher Neural)
- Auto Image Search (Serper)
- Landscape & Portrait format

## Installation
1. Download all files
2. Run `install.bat`
3. Double click desktop shortcut
4. Enter license key
5. Enter API keys
6. Generate videos!
